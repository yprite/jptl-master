## 1. 문제 생성용 단어장

### 프롬프트

```

요구사항:
- JSON 구조: { "N5": [단어 배열] } 형식
- 각 단어는 다음 필드를 포함해야 합니다:
  - "word": 한자/히라가나/가타카나로 된 단어 (예: "こんにちは", "本", "食べる")
  - "meaning": 한국어 뜻 (예: "안녕하세요", "책", "먹다")
  - "reading": 히라가나로 된 읽기 (예: "こんにちは", "ほん", "たべる")
- N5 레벨에 적합한 기본 단어 50개를 생성해주세요
- 다양한 품사 포함 (명사, 동사, 형용사, 부사 등)
- 일상생활에서 자주 사용되는 단어 우선

출력 형식:
- JSON만 출력 (설명 없이)
- 유효한 JSON 형식이어야 함
- 들여쓰기 2칸 사용
```

## 4. 청해 문제

### 프롬프트

```
JLPT N5 레벨의 청해 문제를 JSON 형식으로 생성해주세요.

요구사항:
- JSON 구조: { "N5": [문제 배열] } 형식
- 각 문제는 다음 필드를 포함해야 합니다:
  - "dialogue": 대화 내용 (예: "（会話）\nA: おはようございます。\nB: おはようございます。")
  - "question": 문제 질문 (예: "Bさんは何について話していますか？")
  - "choices": 선택지 배열 (예: ["時間", "天気", "食べ物", "仕事"])
    - 4개의 선택지 필요
  - "correct_answer": 정답 (예: "天気")
  - "explanation": 한국어로 된 해설 (예: "Bさんは「今日はいい天気ですね」と言っています。")
  - "difficulty": 난이도 (1-5 범위, 1이 가장 쉬움)
  - "audio_text": 오디오용 텍스트 (대화 내용을 자연스럽게 연결한 텍스트)
- N5 레벨에 적합한 청해 문제 30개를 생성해주세요
- 일상생활 대화 상황 중심 (인사, 쇼핑, 시간, 장소 등)
- 대화는 자연스럽고 현실적으로 작성
- 난이도는 1-3 범위로 설정 (N5는 초급)
- 해설은 명확하고 이해하기 쉽게 작성

출력 형식:
- JSON만 출력 (설명 없이)
- 유효한 JSON 형식이어야 함
- 들여쓰기 2칸 사용
- 특수문자(줄바꿈 등)는 이스케이프 처리
```

- 난이도는 1-3 범위로 설정 (N5는 초급)
- 해설은 명확하고 이해하기 쉽게 작성
- 지문 길이는 1-3문장으로 제한

출력 형식:
- JSON만 출력 (설명 없이)
- 유효한 JSON 형식이어야 함

## 사용 팁

### 1. 레벨별 요구사항 조정
- **N5**: 가장 기본적인 내용, 난이도 1-3
- **N4**: 기본~중급, 난이도 2-4
- **N3**: 중급, 난이도 3-5
- **N2**: 중상급, 난이도 4-5
- **N1**: 상급, 난이도 5

### 2. 생성 개수 조정
프롬프트에서 "50개", "30개" 등의 숫자를 원하는 개수로 변경할 수 있습니다.

### 3. 다중 레벨 생성
여러 레벨을 한 번에 생성하려면 다음과 같이 요청할 수 있습니다:

```
JLPT N5, N4, N3 레벨의 일본어 단어장을 JSON 형식으로 생성해주세요.

요구사항:
- JSON 구조: { "N5": [...], "N4": [...], "N3": [...] } 형식
- 각 레벨별로 30개씩 생성
- (나머지 요구사항 동일)
```

### 4. 특정 주제로 제한
특정 주제로 제한하려면 프롬프트에 추가:

```
- 주제: 음식, 쇼핑, 여행 등 일상생활 관련 단어만 포함
```

### 5. JSON 검증
생성된 JSON은 반드시 JSON 검증 도구로 확인하세요:
- 온라인: https://jsonlint.com/
- 로컬: `python -m json.tool < file.json`

---

## 주의사항

1. **출력 형식**: ChatGPT가 설명을 추가하지 않도록 "JSON만 출력"을 명시하세요.
2. **특수문자**: 줄바꿈(`\n`) 등 특수문자는 JSON에서 올바르게 이스케이프되어야 합니다.
3. **유효성 검증**: 생성된 JSON은 반드시 파싱 가능한지 확인하세요.
4. **일본어 정확성**: 생성된 일본어 내용의 정확성을 검토하세요.
5. **난이도 일관성**: 같은 레벨의 문제들은 비슷한 난이도를 유지해야 합니다.
